# Cabinet Designer App Changelog

## [Phase 5] - 2026-02-21
### Added
- **Canvas Visuals**: 1px high-contrast thin lines for a precise technical sketch look.
- **Canvas Visuals**: Dashed lines `[4, 4]` for all internal accessories (shelves, internal drawers) and `[2, 4]` for hanging rods.
- **Canvas Visuals**: Full-width kick plates (10cm height) replacing the previous inset style.
- **Editor Controls**: Door width slider allowing doors to be partial width.
- **Editor Controls**: Custom cabinet naming (displays on canvas).
- **Editor Controls**: LED light strip placement selector (top, bottom, left, right).
- **Project Management**: "⭐ 另存新檔" (Save As) button functionality.
- **Renderer Settings**: New Open/Closed door toggle (`doorState`).
- **Renderer Settings**: Added explicit, strict physical constraints to the Gemini prompt to prevent AI hallucinated objects (vases, plants).

## [Phase 1-4] - 2026-02-20
### Added
- 2D canvas drawing with ceilings, spotlights, floor textures.
- Material assignment UI with visual swatches.
- Project save/load integrating Dexie DB.
- White "Glassmorphism" light theme UI overhaul.
- GitHub Actions deployment added.
